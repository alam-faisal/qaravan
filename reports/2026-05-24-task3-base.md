## What was written

### `src/qaravan/core/base.py`

Six classes + two exceptions in one file:

**`Gate(name, indices, matrix, time=None)`**
- `indices` normalised to `list[int]` in `__init__`
- `local_dim` stored as attribute (computed once from matrix shape)
- `num_sites` is a `@property` → `len(self.indices)`
- `dagger()` → new `Gate` with conjugate-transposed matrix and `"†"` suffix
- No `as_superop` — that's Simulator's job at compile time

**`Circuit(gates, n=None, local_dim=2)`**
- `n` inferred from `max(gate.indices) + 1` if not given
- `construct_layers()` — greedy layer packing, mutates `self.layers`
- `decompose()` — stub, raises `NotImplementedError` (implemented in Task 4)
- `dagger()`, `copy()`, `to_matrix()` — all return new `Circuit`; original never mutated
- `+`, `*`, `__rmul__`, `__len__`, `__getitem__`
- `_embed_gate()` helper handles both contiguous and non-contiguous site indices

**`State(ABC)`** — 4 abstract methods: `expectation`, `sample`, `sample_and_collapse`, `overlap`. `rdm` deliberately excluded.

**`Observable(ABC)`** — pure descriptor: `name: str`, `indices: list[int]`. No methods. `State.expectation` dispatches on concrete type.

**`NoiseModel(ABC)`** — `get_kraus` raises `NotImplementedError`; `get_superop` implemented in terms of it; `sample_error` raises `NotImplementedError`. None are `@abstractmethod`.

**`Simulator(ABC)`** — `run()` defined in base: `decompose` (if kwarg) → `construct_layers` → `_insert_noise` → per-gate `translate_gate` + `_apply_translated_gate`. Default `_insert_noise` raises `IncompatibleNoiseError` if `nm is not None`.

**Exceptions:** `IncompatibleNoiseError(Exception)`, `IncompatibleStateError(TypeError)`.
